# Flappy Bird Game

This is a simple Flappy Bird game implemented using HTML, CSS, and JavaScript.

## How to Play

- Press the spacebar to make the bird fly.
- Avoid the pipes and try to get the highest score.

## How to Run

1. Open `index.html` in a web browser.
2. Enjoy the game!
